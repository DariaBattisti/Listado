Listado de 102 etiquetas HTML

1) html

Etiqueta: <html> ... </html>

Descripción: Define el inicio y fin de un documento HTML.

Ejemplo:

  <html>
  <head></head>
  <body></body>
  </html>

Elemento adicional: Es la raíz del documento; dentro van <head> y <body>.

2) head

Etiqueta: <head> ... </head>

Descripción: Contiene metadatos y configuraciones, no se muestra en la página.

Ejemplo:

  <head>
  <title>Mi página</title>
  </head>

Elemento adicional: Aquí van <meta>, <title> y configuraciones.

3) title

Etiqueta: <title> ... </title>

Descripción: Define el título que aparece en la pestaña del navegador.

Ejemplo:

  <title>Mi primera página</title>

Elemento adicional: Importante para SEO y accesibilidad. Solo una por documento.

4) meta

Etiqueta: <meta>

Descripción: Proporciona información sobre el documento (metadatos), como codificación de caracteres o descripción.

Ejemplo:

  <meta charset="UTF-8">
  <meta name="description" content="Ejemplo de etiqueta meta">

Elemento adicional: No tiene cierre; se coloca dentro de <head>.

5) body

Etiqueta: <body> ... </body>

Descripción: Contiene todo el contenido visible de la página.

Ejemplo:

  <body>
  <p>Hola mundo</p>
    </body>

Elemento adicional: Puede tener textos, imágenes, enlaces, tablas, etc.

6) h1

Etiqueta: <h1> ... </h1>

Descripción: Encabezado principal (nivel más alto).

Ejemplo:

  <h1>Bienvenidos a mi página</h1>

Elemento adicional: Solo debería haber un <h1> principal por página.

7) h2

Etiqueta: <h2> ... </h2>

Descripción: Encabezado de segundo nivel.

Ejemplo:

  <h2>Esto es un subtítulo</h2>

Esto es un subtítulo

Elemento adicional: Se usan para estructurar jerárquicamente el contenido.

8) p

Etiqueta: <p> ... </p>

Descripción: Representa un párrafo de texto.

Ejemplo:

<p>Este es un párrafo de ejemplo.</p>

Este es un párrafo de ejemplo.

Elemento adicional: Cada <p> crea automáticamente un espacio debajo.

9) br

Etiqueta: <br>

Descripción: Inserta un salto de línea dentro de un texto.

Ejemplo:

Hola <br> mundo

Hola
mundo

Elemento adicional: Es una etiqueta vacía (no necesita cierre).

10) hr

Etiqueta: <hr>

Descripción: Inserta una línea horizontal para separar secciones.

Ejemplo:

  <p>Sección 1</p>
  <hr>
  <p>Sección 2</p>

Sección 1

Sección 2

Elemento adicional: También es etiqueta vacía.

11) em

Etiqueta: <em> ... </em>

Descripción: Marca énfasis en un texto (se suele mostrar en cursiva).

Ejemplo:

  <em>Texto enfatizado</em>
Texto enfatizado

Elemento adicional: Indica importancia semántica, no solo estilo visual.

12) strong

Etiqueta: <strong> ... </strong>

Descripción: Marca un texto con énfasis fuerte (importancia semántica).

Ejemplo:

  <strong>Texto importante</strong>
Texto importante

Elemento adicional: Normalmente se muestra en negrita.

13) b

Etiqueta: <b> ... </b>

Descripción: Pone el texto en negrita sin indicar importancia.

Ejemplo:

  <b>Texto en negrita</b>
Texto en negrita

Elemento adicional: Es solo estilo visual, no semántico.

14) i

Etiqueta: <i> ... </i>

Descripción: Muestra el texto en cursiva sin indicar énfasis.

Ejemplo:

  <i>Texto en cursiva</i>
Texto en cursiva

Elemento adicional: Se usa en nombres, términos extranjeros o estilos.

15) u

Etiqueta: <u> ... </u>

Descripción: Subraya el texto.

Ejemplo:

  <u>Texto subrayado</u>
Texto subrayado

Elemento adicional: Hoy se usa poco porque puede confundirse con enlaces.

16) mark

Etiqueta: <mark> ... </mark>

Descripción: Resalta un texto como si estuviera marcado.

Ejemplo:

  <mark>Texto resaltado</mark>
Texto resaltado

Elemento adicional: Normalmente aparece en amarillo por defecto.

17) small

Etiqueta: <small> ... </small>

Descripción: Muestra el texto en un tamaño más pequeño.

Ejemplo:

  <small>Nota pequeña</small>
Nota pequeña

Elemento adicional: Útil para aclaraciones o notas.

18) sub

Etiqueta: <sub> ... </sub>

Descripción: Muestra texto en subíndice (debajo de la línea).

Ejemplo:

  H<sub>2</sub>O
H2O

Elemento adicional: Muy usado en fórmulas químicas.

19) sup

Etiqueta: <sup> ... </sup>

Descripción: Muestra texto en superíndice (arriba de la línea).

Ejemplo:

  x<sup>2</sup>
x2

Elemento adicional: Muy usado en matemáticas y referencias.

20) abbr

Etiqueta: <abbr> ... </abbr>

Descripción: Define una abreviatura o acrónimo, mostrando su significado.

Ejemplo:

  <abbr title="Organización de las Naciones Unidas">ONU</abbr>
UASD

Elemento adicional: Al pasar el ratón, aparece la explicación.

21) pre

Etiqueta: <pre> ... </pre>

Descripción: Muestra texto preformateado (respeta espacios y saltos de línea).

Ejemplo:

  <pre>
  Línea 1
    Línea 2 con espacios
  Línea 3
  </pre>
  
  Línea 1
    Línea 2 con espacios
  Línea 3

Elemento adicional: Útil para mostrar código o texto alineado.

22) cite

Etiqueta: <cite> ... </cite>

Descripción: Indica el título de una obra, libro, película u otro recurso citado.

Ejemplo:

  <p>Mi libro favorito es <cite>Sandokán</cite>.</p>

Mi libro favorito es Sandokán.

Elemento adicional: Normalmente se muestra en cursiva.

23) q

Etiqueta: <q> ... </q>

Descripción: Representa una cita corta en línea.

Ejemplo:

  <p>Ella dijo: <q> La vida es corta</q>.</p>

Ella dijo: La vida es corta.

Elemento adicional: El navegador añade comillas automáticamente.

24) blockquote

Etiqueta: <blockquote> ... </blockquote>

Descripción: Representa una cita larga o en bloque.

Ejemplo:

  <blockquote>
  Si un niño no puede aprender de la forma en que enseñamos, quizás deberíamos enseñar como él aprende.
  </blockquote>
Si un niño no puede aprender de la forma en que enseñamos, quizás deberíamos enseñar como él aprende.

Elemento adicional: Se suele mostrar con sangría.

25) code

Etiqueta: <code> ... </code>

Descripción: Muestra un fragmento de código en línea.

Ejemplo:

  <p>El comando <code>print()</code> se usa en Python.</p>

El comando print() se usa en Python.

Elemento adicional: Se muestra en fuente monoespaciada.

26) kbd

Etiqueta: <kbd> ... </kbd>

Descripción: Representa la entrada de teclado que debe escribir el usuario.

Ejemplo:

  <p>Presiona <kbd>Ctrl</kbd> + <kbd>C</kbd> para copiar.</p>

Presiona Ctrl + C para copiar.

Elemento adicional: Útil para atajos de teclado.

27) samp

Etiqueta: <samp> ... </samp>

Descripción: Representa la salida de un programa o sistema.

Ejemplo:

  <p>Resultado: <samp>Archivo no encontrado</samp></p>

Resultado: Archivo no encontrado

Elemento adicional: Suele mostrarse en monoespaciada como <code>.

28) var

Etiqueta: <var> ... </var>

Descripción: Representa el nombre de una variable en matemáticas o programación.

Ejemplo:

  <p>La fórmula es: <var>x</var> + 2 = 10</p>

La fórmula es: x + 2 = 10

Elemento adicional: Se suele mostrar en cursiva.

29) time

Etiqueta: <time> ... </time>

Descripción: Representa una fecha u hora.

Ejemplo:

  <p>Publicado el <time datetime="2025-09-15">15 de septiembre de 2025</time></p>

Publicado el 15 de septiembre de 2025

Elemento adicional: Útil para máquinas y SEO.

30) dfn

Etiqueta: <dfn> ... </dfn>

Descripción: Señala la definición de un término.

Ejemplo:

  <p><dfn>HTML</dfn> es el lenguaje de marcado para crear páginas web.</p>

HTML es el lenguaje de marcado para crear páginas web.

Elemento adicional: Normalmente se muestra en cursiva.

31) ul

Etiqueta: <ul> ... </ul>

Descripción: Crea una lista desordenada (con viñetas).

Ejemplo:

  <ul>
  <li>Manzana</li>
  <li>Pera</li>
  <li>Banana</li>
  </ul>
  • Manzana
  • Pera
  • Banana

Elemento adicional: Cada elemento debe ir dentro de <li>.

32) ol

Etiqueta: <ol> ... </ol>

Descripción: Crea una lista ordenada (numerada).

Ejemplo:

  <ol>
  <li>Primero</li>
  <li>Segundo</li>
  <li>Tercero</li>
  </ol>
  1. Primero
  2. Segundo
  3. Tercero

Elemento adicional: Se puede cambiar el estilo de numeración con atributos.

33) li

Etiqueta: <li> ... </li>

Descripción: Define cada elemento dentro de una lista (<ul> o <ol>).

Ejemplo:

  <ul>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
  </ul>
  • Elemento 1
  • Elemento 2

Elemento adicional: Puede contener texto, imágenes u otras etiquetas.

34) dl

Etiqueta: <dl> ... </dl>

Descripción: Define una lista de definiciones.

Ejemplo:

  <dl>
  <dt>HTML</dt>
  <dd>Lenguaje de marcado de hipertexto</dd>
  </dl>
HTML
Lenguaje de marcado de hipertexto

Elemento adicional: Se compone de términos (<dt>) y descripciones (<dd>).

35) dt

Etiqueta: <dt> ... </dt>

Descripción: Representa el término dentro de una lista de definiciones.

Ejemplo:

  <dl>
  <dt>CSS</dt>
  <dd>Lenguaje de estilos en cascada</dd>
  </dl>
CSS
Lenguaje de estilos en cascada

Elemento adicional: Normalmente aparece en negrita.

36) dd

Etiqueta: <dd> ... </dd>

Descripción: Define la descripción de un término en una lista de definiciones.

Ejemplo:

  <dl>
  <dt>JavaScript</dt>
  <dd>Lenguaje de programación para la web</dd>
  </dl>
JavaScript
Lenguaje de programación para la web

Elemento adicional: Suele mostrarse con sangría respecto al término.

37) a

Etiqueta: <a> ... </a>

Descripción: Crea un enlace o hipervínculo a otra página, archivo o sección.

Ejemplo:

  <a href="https://www.google.com">Ir a Google</a>
Ir a Google

Elemento adicional: Con el atributo target="_blank" abre el enlace en otra pestaña.

38) img

Etiqueta: <img>

Descripción: Inserta una imagen en la página.

Ejemplo:

  <img src="foto.jpg" alt="Descripción de la imagen">

Elemento adicional: Es una etiqueta vacía, siempre debe incluir el atributo alt para accesibilidad.

39) figure

Etiqueta: <figure> ... </figure>

Descripción: Contenedor para imágenes, ilustraciones o gráficos.

Ejemplo:

  <figure>
  <img src="sandia.jpg" alt="Sandía">
  </figure>

Elemento adicional: Se suele usar junto a <figcaption>.

40) figcaption

Etiqueta: <figcaption> ... </figcaption>

Descripción: Añade un pie de foto o título a <figure>.

Ejemplo:

  <figure>
  <img src="sandia.jpg" alt="Sandía">
  <figcaption>Mano sujetando sandía</figcaption>
  </figure>

Elemento adicional: Debe ir dentro de <figure>.

41) table

Etiqueta: <table> ... </table>

Descripción: Crea una tabla para organizar datos en filas y columnas.

Ejemplo:

  <table border="1">
  <tr><td>A</td><td>B</td></tr>
  </table>
AB

Elemento adicional: Contiene <tr>, <td> y <th>.

42) tr

Etiqueta: <tr> ... </tr>

Descripción: Define una fila dentro de una tabla.

Ejemplo:

  <tr>
  <td>Uno</td>
  <td>Dos</td>
  </tr>
Uno Dos

Elemento adicional: Siempre se usa dentro de <table>.

43) td

Etiqueta: <td> ... </td>

Descripción: Define una celda de datos dentro de una fila.

Ejemplo:

  <tr>
  <td>Lunes</td>
  <td>Martes</td>
  </tr>
Lunes Martes

Elemento adicional: Forma parte del cuerpo de la tabla.

44) th

Etiqueta: <th> ... </th>

Descripción: Define una celda de encabezado dentro de una tabla.

Ejemplo:

  <table border="1">
  <tr>
    <th>Nombre</th>
    <th>Edad</th>
  </tr>
  </table>
Nombre Edad

Elemento adicional: El texto se muestra en negrita y centrado por defecto.

45) thead

Etiqueta: <thead> ... </thead>

Descripción: Agrupa el encabezado de una tabla.

Ejemplo:

  <table border="1">
  <thead>
    <tr><th>Producto</th><th>Precio</th></tr>
  </thead>
  </table>
ProductoPrecio

Elemento adicional: Se usa junto a <tbody> y <tfoot>.

46) tbody

Etiqueta: <tbody> ... </tbody>

Descripción: Contiene el cuerpo principal de una tabla.

Ejemplo:

  <table border="1">
  <tbody>
    <tr><td>Pantalón</td><td>200</td></tr>
    <tr><td>Falda</td><td>100</td></tr>
  </tbody>
  </table>
Pantalón200
Falda100

Elemento adicional: Puede haber varios <tbody> en una misma tabla.

47) tfoot

Etiqueta: <tfoot> ... </tfoot>

Descripción: Representa el pie de una tabla, usado para totales o notas.

Ejemplo:

  <table border="1">
  <tfoot>
    <tr><td colspan="2">Total: 300</td></tr>
  </tfoot>
  </table>
Total: 300

Elemento adicional: Aunque se declara al inicio, se muestra al final de la tabla.

48) caption

Etiqueta: <caption> ... </caption>

Descripción: Añade un título o descripción a la tabla.

Ejemplo:

  <table border="1">
  <caption>Lista de precios</caption>
  <tr><td>Producto</td><td>Precio</td></tr>
  </table>
Lista de precios
ProductoPrecio

Elemento adicional: Se coloca justo después de la apertura de <table>.

49) col

Etiqueta: <col>

Descripción: Define propiedades para una columna de una tabla.

Ejemplo:

  <table border="1">
  <colgroup>
    <col style="background-color:yellow">
  </colgroup>
  <tr><td>1</td><td>2</td></tr>
  </table>
12

Elemento adicional: No necesita cierre; se usa dentro de <colgroup>.

50) colgroup

Etiqueta: <colgroup> ... </colgroup>

Descripción: Agrupa una o varias columnas para aplicar estilos en bloque.

Ejemplo:

  <table border="1">
  <colgroup>
    <col style="background-color:lightblue">
    <col style="background-color:lightgreen">
  </colgroup>
  <tr><td>1</td><td>2</td></tr>
  </table>
12

Elemento adicional: Permite dar estilos por columna, no solo por celda.

51) form

Etiqueta: <form> ... </form>

Descripción: Crea un formulario para recopilar datos del usuario.

Ejemplo:

  <form action="/procesar" method="post">
  <input type="text" name="nombre">
  </form>

Elemento adicional: Usa atributos action y method para definir cómo se envían los datos.

52) input

Etiqueta: <input>

Descripción: Campo de entrada; puede ser texto, contraseña, número, etc.

Ejemplo:

  <input type="email" name="correo" placeholder="Ingresa tu correo">

Elemento adicional: Es una etiqueta vacía y cambia con el atributo type.

53) label

Etiqueta: <label> ... </label>

Descripción: Asocia un texto descriptivo a un campo de formulario.

Ejemplo:

  <label for="usuario">Nombre:</label>
  <input id="usuario" type="text">

Elemento adicional: El atributo for debe coincidir con el id del campo.

54) textarea

Etiqueta: <textarea> ... </textarea>

Descripción: Crea un área de texto de varias líneas.

Ejemplo:

  <textarea rows="4" cols="30">Escribe aquí...</textarea>

Elemento adicional: Se usa para comentarios o mensajes largos.

55) button

Etiqueta: <button> ... </button>

Descripción: Crea un botón en el formulario.

Ejemplo:

  <button type="submit">Enviar</button>

Elemento adicional: Puede ser submit, reset, button...

56) select

Etiqueta: <select> ... </select>

Descripción: Crea un menú desplegable de opciones.

Ejemplo:

  <select name="fruta">
  <option>Femenino</option>
  <option>Masculino</option>
  </select>

Elemento adicional: Se combina con <option> para listar opciones.

57) option

Etiqueta: <option> ... </option>

Descripción: Define una opción dentro de un menú desplegable.

Ejemplo (en código):

  <select>
  <option value="1">Opción 1</option>
  <option value="2" selected>Opción 2 (por defecto)</option>
  </select>

Elemento adicional: Usa selected para marcar la opción por defecto.

58) optgroup

Etiqueta: <optgroup> ... </optgroup>

Descripción: Agrupa opciones relacionadas dentro de un menú desplegable.

Ejemplo:

  <select>
  <optgroup label="Género">
    <option>Femenino</option>
    <option>Masculino</option>
  </optgroup>
  </select>

Elemento adicional: Se usa para organizar mejor listas largas.

59) fieldset

Etiqueta: <fieldset> ... </fieldset>

Descripción: Agrupa campos relacionados en un formulario.

Ejemplo:

  <fieldset>
  <input type="text" placeholder="Nombre">
  </fieldset>

Elemento adicional: Dibuja un borde alrededor de los campos agrupados.

60) legend

Etiqueta: <legend> ... </legend>

Descripción: Define un título para un grupo de campos (<fieldset>).

Ejemplo:

  <fieldset>
  <legend>Información de contacto</legend>
  <input type="email" placeholder="Nombre">
  </fieldset>
Datos personales

Elemento adicional: Se muestra como título dentro del borde de <fieldset>.

61) datalist

Etiqueta: <datalist> ... </datalist>

Descripción: Proporciona una lista de sugerencias para un campo de entrada.

Ejemplo:

  <input list="frutas">
  <datalist id="frutas">
  <option value="Manzana">
  <option value="Pera">
  <option value="Banana">
  </datalist>

Elemento adicional: Se usa junto a <input list="">.

62) output

Etiqueta: <output> ... </output>

Descripción: Muestra el resultado de un cálculo o acción.

Ejemplo:

  <form oninput="resultado.value=parseInt(a.value)+parseInt(b.value)">
  <input type="number" id="a" value="2"> +
  <input type="number" id="b" value="3"> =
  <output name="resultado">5</output>
  </form>
+ = 5

Elemento adicional: Se actualiza dinámicamente con JavaScript o atributos.

63) progress

Etiqueta: <progress> ... </progress>

Descripción: Representa una barra de progreso.

Ejemplo:

  <progress value="70" max="100">70%</progress>
70%

Elemento adicional: El atributo value indica el progreso actual.

64) meter

Etiqueta: <meter> ... </meter>

Descripción: Representa una medida dentro de un rango.

Ejemplo:

  <meter value="0.6" min="0" max="1">60%</meter>
60%

Elemento adicional: Útil para mostrar valores como batería o calificaciones.

65) audio

Etiqueta: <audio> ... </audio>

Descripción: Inserta un reproductor de audio.

Ejemplo:

  <audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  </audio>

Elemento adicional: El atributo controls añade botones de reproducción.

66) video

Etiqueta: <video> ... </video>

Descripción: Inserta un reproductor de video.

Ejemplo:

  <video controls width="300">
  <source src="video.mp4" type="video/mp4">
  </video>

Elemento adicional: Se pueden ajustar tamaño, controles y varias fuentes.

67) source

Etiqueta: <source>

Descripción: Define una fuente alternativa para <audio> o <video>.

Ejemplo:

  <video controls>
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
  </video>

Elemento adicional: El navegador usa la primera fuente compatible.

68) embed

Etiqueta: <embed>

Descripción: Inserta contenido externo (ej. PDF o Flash).

Ejemplo:

  <embed src="documento.pdf" type="application/pdf" width="400" height="300">

Elemento adicional: Es etiqueta vacía y permite incrustar distintos tipos de archivos.

69) object

Etiqueta: <object> ... </object>

Descripción: Inserta un recurso externo, como imágenes, documentos o animaciones.

Ejemplo:

  <object data="documento.pdf" type="application/pdf" width="400" height="300"></object>

Elemento adicional: Puede contener texto alternativo dentro por si el recurso no se carga.

70) iframe

Etiqueta: <iframe> ... </iframe>

Descripción: Inserta otra página web dentro de la página actual.

Ejemplo:

  <iframe src="https://es.wikipedia.org/wiki/HTML" width="400" height="300"></iframe>

Elemento adicional: No todas las páginas permiten iframes.

71) header

Etiqueta: <header> ... </header>

Descripción: Representa la cabecera de una página o sección.

Ejemplo:

  <header>
  <h1>Mi sitio web</h1>
  </header>

Mi sitio web

Elemento adicional: Puede contener logotipo, menú o título principal.

72) nav

Etiqueta: <nav> ... </nav>

Descripción: Define una sección de navegación con enlaces.

Ejemplo:

  <nav>
  <a href="#inicio">Inicio</a> |
  <a href="#contacto">Contacto</a>
  </nav>
Inicio | Contacto

Elemento adicional: Ayuda a los buscadores y lectores de pantalla a identificar menús.

73) main

Etiqueta: <main> ... </main>

Descripción: Representa el contenido principal de la página.

Ejemplo:

  <main>
  <p>Aquí va el contenido más importante.</p>
  </main>

Aquí va el contenido más importante.

Elemento adicional: Solo debe haber un <main> por página.

74) section

Etiqueta: <section> ... </section>

Descripción: Define una sección temática dentro de un documento.

Ejemplo:

  <section>
  <h2>Noticias</h2>
  <p>Últimas actualizaciones del sitio.</p>
  </section>

Noticias

Últimas actualizaciones del sitio.

Elemento adicional: Se usa para organizar el contenido en bloques.

75) article

Etiqueta: <article> ... </article>

Descripción: Contenido independiente como una noticia o publicación.

Ejemplo:

  <article>
  <h2>Artículo 1</h2>
  <p>Este es el contenido del artículo.</p>
  </article>

Artículo 1

Este es el contenido del artículo.

Elemento adicional: Tiene sentido por sí mismo, incluso fuera de la página.

76) aside

Etiqueta: <aside> ... </aside>

Descripción: Define contenido complementario o lateral.

Ejemplo:

  <aside>
  <p>Publicidad o enlaces relacionados</p>
  </aside>

Publicidad o enlaces relacionados

Elemento adicional: No es parte central del contenido, sino apoyo.

77) footer

Etiqueta: <footer> ... </footer>

Descripción: Define el pie de página de un documento o sección.

Ejemplo:

  <footer>
  <p>© 2025 Mi sitio web</p>
  </footer>

© 2025 Mi sitio web

Elemento adicional: Puede contener autoría, enlaces o contacto.

78) address

Etiqueta: <address> ... </address>

Descripción: Representa información de contacto.

Ejemplo:

  <address>
  Escrito por Daria Battisti<br>
  Santo Domingo, RD
  </address>
Escrito por Daria Battisti
Santo Domingo, RD

Elemento adicional: Suele mostrarse en cursiva.

79) div

Etiqueta: <div> ... </div>

Descripción: Contenedor genérico en bloque para agrupar elementos.

Ejemplo:

  <div>
  <p>Texto dentro de un div</p>
  </div>

Texto dentro de un div

Elemento adicional: Se usa mucho para maquetación y diseño.

80) span

Etiqueta: <span> ... </span>

Descripción: Contenedor en línea para aplicar estilos o agrupar texto.

Ejemplo:

  <p>Este es un <span>texto resaltado</span></p>

Este es un texto resaltado

Elemento adicional: No rompe la línea, se usa dentro de un párrafo.

81) style

Etiqueta: <style> ... </style>

Descripción: Permite escribir reglas de CSS dentro del documento HTML.

Ejemplo:

  <style>
  p { color:  rgb(41, 39, 39); }
  </style>

Elemento adicional: Va dentro de <head> y afecta a toda la página.

82) link

Etiqueta: <link>

Descripción: Vincula archivos externos, como hojas de estilo CSS o íconos.

Ejemplo:

  <link rel="stylesheet" href="estilos.css">

Elemento adicional: Siempre va dentro de <head> y no tiene cierre.

83) script

Etiqueta: <script> ... </script>

Descripción: Inserta o enlaza código JavaScript.

Ejemplo:

  <script>
  alert("Bienvenido");
  </script>

Elemento adicional: Puede ir en <head> o al final de <body>.

84) noscript

Etiqueta: <noscript> ... </noscript>

Descripción: Muestra contenido alternativo si JavaScript está deshabilitado.

Ejemplo:

  <noscript>Tu navegador no soporta JavaScript.</noscript>

Elemento adicional: Se usa como mensaje de respaldo.

85) details

Etiqueta: <details> ... </details>

Descripción: Crea un cuadro desplegable que se puede expandir o contraer.

Ejemplo:

  <details>
  <summary>Ver más</summary>
  <p>Texto oculto.</p>
  </details>
Ver más

Texto oculto.

Elemento adicional: El atributo open lo muestra expandido por defecto.

86) summary

Etiqueta: <summary> ... </summary>

Descripción: Define el título visible de un bloque <details>.

Ejemplo:

  <details>
  <summary>Información</summary>
  <p>Texto oculto</p>
  </details>
Información

Texto oculto

Elemento adicional: Siempre debe estar dentro de <details>.

87) dialog

Etiqueta: <dialog> ... </dialog>

Descripción: Define un cuadro de diálogo o ventana emergente.

Ejemplo:

  <dialog open>Este es un cuadro de diálogo</dialog>
Este es un cuadro de diálogo

Elemento adicional: Se puede abrir o cerrar con JavaScript.

88) canvas

Etiqueta: <canvas> ... </canvas>

Descripción: Se usa para dibujar gráficos dinámicos con JavaScript.

Ejemplo:

  <canvas id="miCanvas" width="200" height="100"></canvas>

Elemento adicional: Por sí solo no muestra nada, necesita JS para dibujar.

89) svg

Etiqueta: <svg> ... </svg>

Descripción: Define gráficos vectoriales escalables.

Ejemplo:

  <svg width="100" height="100">
  <circle cx="50" cy="50" r="40" fill="blue"></circle>
  </svg>

Elemento adicional: Permite dibujar formas como círculos, rectángulos y líneas.

90) title (en SVG)

Etiqueta: <title> ... </title>

Descripción: Añade un título descriptivo a un gráfico SVG.

Ejemplo:

  <svg width="100" height="100">
  <title>Círculo azul</title>
  <circle cx="50" cy="50" r="40" fill="blue"></circle>
  </svg>

Elemento adicional: Al pasar el ratón sobre el SVG, aparece como tooltip.

91) data

Etiqueta: <data> ... </data>

Descripción: Asocia un valor legible para humanos con un valor de máquina.

Ejemplo:

  <p>Precio: <data value="50">50 dólares</data></p>

Precio: 50 dólares

Elemento adicional: Útil para aplicaciones que procesan datos automáticamente.

92) picture

Descripción: Contenedor de imágenes, permite definir diferentes versiones según el dispositivo, pero también puede usarse con una sola imagen.

Etiqueta: <picture> ... </picture>

Ejemplo:

  <picture>
  <img src="foto.jpg" alt="Ejemplo de imagen" width="300">
  </picture>

Elemento adicional: No es obligatorio usar .webp; también funciona con .jpg o .png.

93) source (en picture)

Etiqueta: <source>

Descripción: Define una fuente de imagen dentro de <picture>. El navegador elige la más adecuada.

Ejemplo:

  <picture>
  <source srcset="foto.jpg" type="image/jpeg">
  <img src="foto.jpg" alt="Ejemplo de imagen" width="300">
  </picture>

Elemento adicional: Si el navegador no reconoce <source>, siempre mostrará la imagen del <img>.

94) base

Etiqueta: <base>

Descripción: Define la URL base para los enlaces relativos.

Ejemplo:

  <head>
  <base href="https://www.ejemplo.com/">
  </head>

Elemento adicional: Solo puede haber uno por documento.

95) s

Etiqueta: <s> ... </s>

Descripción: Muestra texto tachado (sin indicar eliminación semántica).

Ejemplo:

  <p>Antes: <s>$100</s> Ahora: $80</p>

Antes: $100 Ahora: $80

Elemento adicional: Diferente de <del>, que sí indica contenido eliminado.

96) bdi

Etiqueta: <bdi> ... </bdi>

Descripción: Aísla un texto que puede tener dirección de escritura distinta.

Ejemplo:

  <p>Usuario: <bdi>مرحبا</bdi></p>

Usuario: مرحبا

Elemento adicional: Útil para mezclar idiomas en el mismo párrafo.

97) bdo

Etiqueta: <bdo> ... </bdo>

Descripción: Fuerza la dirección de un texto.

Ejemplo:

  <bdo dir="rtl">Texto al revés</bdo>
Texto al revés

Elemento adicional: El atributo dir acepta "ltr" o "rtl".

98) wbr

Etiqueta: <wbr>

Descripción: Indica un posible salto de línea en palabras largas.

Ejemplo:

  <p>supercalifragilistico<wbr>espialidoso</p>

supercalifragilisticoespialidoso

Elemento adicional: No se ve, pero ayuda a cortar texto largo sin romper diseño.

99) template

Etiqueta: <template> ... </template>

Descripción: Contiene contenido oculto que se activa con JavaScript.

Ejemplo:

  <template id="tarjeta">
  <p>Contenido oculto</p>
  </template>

Elemento adicional: No se muestra en pantalla hasta que se usa con JS.

100) slot

Etiqueta: <slot> ... </slot>

Descripción: Espacio para contenido en Web Components.

Ejemplo:

  <my-element>
  <span slot="titulo">Título dinámico</span>
  </my-element>
Título dinámico

Elemento adicional: Permite personalizar componentes.

101) ins

Etiqueta: <ins> ... </ins>

Descripción: Representa texto insertado en un documento.

Ejemplo:

  <p>Este es un <ins>text añadido</ins></p>

Este es un texto añadido

Elemento adicional: Normalmente aparece subrayado.

102) del

Etiqueta: <del> ... </del>

Descripción: Representa texto eliminado.

Ejemplo:

  <p>Precio: <del>$50</del> ahora $40</p>

Precio: $50 ahora $40

Elemento adicional: Normalmente se muestra tachado.
